# NeMo Automodel Documentation

Welcome to the comprehensive documentation for NVIDIA NeMo Automodel - the premier platform for efficient, scalable, and production-ready fine-tuning of large language models, vision language models, and advanced omni-modal architectures.

This documentation is organized to get you productive quickly, whether you're running your first fine-tuning job or architecting enterprise-scale training infrastructure.

## How to Navigate This Documentation

This site is structured around your journey - from initial setup to advanced production deployments. Each section builds upon previous knowledge while remaining accessible for targeted learning.

### Start Here

**New to NeMo Automodel?** Begin with {doc}`About NeMo Automodel <about/index>` to understand the platform, then move to {doc}`Get Started with NeMo Automodel <get-started/index>` for installation and your first training run.

**Ready to Train?** Jump directly to {doc}`About NeMo Automodel Guides <guides/index>` for comprehensive training workflows, or browse {doc}`About Learning Resources <learning-resources/index>` for tutorials and examples.

**Need Technical Details?** Access the complete {doc}`API Reference <api-docs/index>` and {doc}`About NeMo Automodel References <references/index>` for in-depth technical specifications.

### Documentation Structure

Each major section serves a specific purpose in your learning journey:

- **{doc}`Get Started with NeMo Automodel <get-started/index>`**: Installation, environment setup, and first training job
- **{doc}`About NeMo Automodel Guides <guides/index>`**: Step-by-step training workflows for different model types and scenarios  
- **{doc}`About Learning Resources <learning-resources/index>`**: Hands-on tutorials, examples, and real-world use cases
- **{doc}`About NeMo Automodel Model Coverage <model-coverage/index>`**: Supported architectures and compatibility matrices
- **{doc}`About NeMo Automodel References <references/index>`**: Technical specifications, CLI commands, and API documentation

### Quick Navigation Tips

- Use the **search function** to find specific topics, error messages, or configuration options
- Check **{doc}`About NeMo Automodel Model Coverage <model-coverage/index>`** to verify support for your target model before starting
- Bookmark **{doc}`About NeMo Automodel References <references/index>`** for quick access to CLI commands and configuration parameters during development
- Visit **{doc}`API Reference <api-docs/index>`** for complete autogenerated API documentation
