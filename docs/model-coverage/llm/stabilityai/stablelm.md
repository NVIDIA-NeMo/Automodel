# StableLM

[StableLM](https://stability.ai/stable-lm) is Stability AI's series of open language models, available in base and instruction-tuned variants across multiple sizes.

## Available Models

- **stablelm-3b-4e1t**: 3B
- **stablelm-base-alpha-7b-v2**: 7B

## Architecture

- `StableLmForCausalLM`
