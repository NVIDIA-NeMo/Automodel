# OLMo

[OLMo](https://allenai.org/olmo) (Open Language Model) is Allen AI's fully open language model — open weights, open training data, and open training code. OLMo-1B and OLMo-7B are trained on Dolma.

## Available Models

- **OLMo-7B-hf**: 7B
- **OLMo-1B-hf**: 1B

## Architecture

- `OLMoForCausalLM`
